About the Project

This project uses climate data collected from the National Center for Hydrology and Meteorology (NCHM) to analyse historical rainfall and temperature patterns in five southern Dzongkhags of Bhutan. The aim is to provide a clear dashboard for understanding climate trends, extreme events, and projected changes from 2026–2055 to support climate awareness, planning, and decision-making.

Data and Methodology

📊

Historical Data Analysis

Historical climate data spanning 1996-2025 is analyzed using Mann-Kendall trend analysis and Sen's slope estimator to quantify significant trends in rainfall and temperature patterns.

📈

Trend Analysis

  • Mann-Kendall Test: Detects monotonic trends in time series data
  • Sen's Slope: Provides non-parametric estimation of trend magnitude
  • p-value: Indicates statistical significance of detected trends
🌧️

Climate Projections

Future climate patterns (2026-2055) are projected using the CNN-BiLSTM (Convolutional Neural Network - Bidirectional Long Short-Term Memory) deep learning model, combining convolutional feature extraction with recurrent neural network capabilities for robust climate forecasting.

🎯

Model Performance Evaluation

The CNN-BiLSTM model delivers reliable climate forecasts across Southern Bhutan by effectively learning complex rainfall and temperature patterns from historical data. Its strong predictive performance supports accurate climate projections, helping users better understand future rainfall and temperature trends across the five study dzongkhags.
